üìö Lessons Learned

Technical Lessons

  • Azure CLI with SSL proxy requires AZURE_CLI_DISABLE_CONNECTION_VERIFICATION=1
  • PowerShell outputs UTF-8 BOM by default - Python must use utf-8-sig
  • Azure CLI command names are inconsistent (secure-scores vs assessment)
  • Azure returns duplicate JSON keys in some APIs - requires regex parsing
  • Subscription context must be explicit in all sub-resource queries
  • Python is py not python in this environment

Process Lessons

  • Testing commands manually first saves debugging time later
  • Progress indicators critical for long-running scripts
  • Incremental transformation (one category at a time) better than all-at-once
  • Team-specific dashboards more actionable than one mega-report
  • Ephemeral C: drive requires daily SharePoint save/restore workflow
